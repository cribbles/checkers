Checkers
========

Summary
-------
Checkers is a Ruby clone of the famous [Checkers](https://en.wikipedia.org/wiki/Checkers) board game. Play against a human or computer opponent, or watch two computer opponents face off against each other.

Usage
-----
```
./checkers.rb
```
